Características Principales

1. Sincronización en Tiempo Real con Google Sheets

La aplicación no requiere una base de datos compleja; utiliza tu documento de Google Sheets como "backend". Al abrir la app, se conecta automáticamente para descargar la lista actualizada de camiones pendientes.

2. Consulta de Información (Solo Lectura)

Selector de Unidades: Cuenta con un menú desplegable para seleccionar el camión o unidad que se va a procesar.

Datos Automáticos: Al seleccionar un camión, la app muestra inmediatamente a qué "Área" pertenece y la cantidad de "Piezas" esperadas. Estos campos están bloqueados (solo lectura) para evitar que el operador los modifique por error.
